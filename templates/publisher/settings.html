⚙️ إعدادات الناشر

1 بيانات تطبيق فيسبوك 🔗 developers.facebook.com →

اذهب إلى Facebook for Developers → My Apps → اختر تطبيقك → Settings → Basic

2 User Access Token 🔗 Graph API Explorer →

من Graph API Explorer → اختر تطبيقك → اضغط "Generate Access Token" → وافق على الصلاحيات التالية:

pages_show_list pages_read_engagement pages_manage_posts pages_read_user_content

يُحفَظ مشفراً في قاعدة البيانات ولا يظهر مجدداً
