# المساهمة في Finora

شكراً لاهتمامك بالمشروع.

## كيفية المساهمة

1. **Fork** المستودع وأنشئ فرعاً من `main` لعملك.
2. التزم بأسلوب الكود الحالي (Python/Flask، تعليقات عربية أو إنجليزية حسب السياق).
3. تأكد من تشغيل الاختبارات محلياً قبل الـ push.
4. افتح **Pull Request** إلى الفرع `main` مع وصف واضح للتغيير.

## تشغيل الاختبارات

```bash
pip install -r requirements-test.txt
pytest
```

## الإبلاغ عن مشاكل أو اقتراحات
