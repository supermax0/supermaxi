## 🎯 الحسابات المحاسبية الرئيسية

| الكود | الحساب | النوع | التعديل |
|------|--------|------|---------|
| `3001` | رأس المال | حقوق ملكية | ✅ فقط عند: الإضافة، السحب، إغلاق الأرباح |
| `1001` | النقدية | أصل | ✅ يزيد عند: المبيعات النقدية، تحصيل الديون، إضافة رأس مال |
| `1101` | المخزون | أصل | ✅ يزيد عند: الشراء، ✅ ينقص عند: البيع (COGS) |
| `1201` | الذمم المدينة | أصل | ✅ يزيد عند: البيع الآجل، ✅ ينقص عند: تحصيل الدين |
| `4001` | إيراد المبيعات | إيراد | ✅ يزيد عند: البيع (نقدي أو آجل) |
| `5001` | تكلفة البضاعة المباعة | مصروف | ✅ يزيد عند: البيع (بقيمة COGS) |
| `6001` | المصاريف | مصروف | ✅ يزيد عند: تسجيل مصروف |

## 🔑 المبادئ الأساسية

### ✅ صحيح:

1. **رأس المال** - يُعدل فقط عند:
   - إضافة رأس مال ✅
   - سحب رأس مال ✅
   - إغلاق الأرباح (نهاية الفترة) ✅

2. **النقدية** - حساب مستقل:
   - جميع التحصيلات النقدية تُضاف ✅
   - جميع المدفوعات تُخصم ✅

3. **المخزون** - أصل:
   - لا يدخل ضمن رأس المال ✅
   - يُخفض عند البيع بقيمة COGS ✅

4. **المبيعات** - إيراد:
   - البيع النقدي: يُضاف للنقدية مباشرة ✅
   - البيع الآجل: يُسجل في الذمم المدينة ✅

5. **الأرباح**:
   - الربح = الإيرادات - COGS - المصاريف ✅
   - لا يُضاف لرأس المال مباشرة ✅
   - يُضاف لرأس المال فقط في نهاية الفترة ✅

## 📝 القيود المحاسبية (Double Entry)

### مثال: بيع نقدي بقيمة 50,000 د.ع، تكلفة 30,000 د.ع

**القيد 1: تسجيل الإيراد**
```
مدين: النقدية (1001) - 50,000 د.ع
دائن: إيراد المبيعات (4001) - 50,000 د.ع
```

**القيد 2: تسجيل COGS**
```
مدين: تكلفة البضاعة المباعة (5001) - 30,000 د.ع
دائن: المخزون (1101) - 30,000 د.ع
```

**النتيجة:**
- النقدية: +50,000 د.ع
- إيراد المبيعات: +50,000 د.ع
- COGS: +30,000 د.ع
- المخزون: -30,000 د.ع
- **الربح من العملية:** 20,000 د.ع (لا يُضاف لرأس المال مباشرة)
