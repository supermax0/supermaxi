## 🔑 القواعد المحاسبية المطبقة

### ✅ 1. فصل المفاهيم المحاسبية

| المفهوم | الحساب | كيف يُحسب |
|---------|--------|-----------|
| **الإيرادات** | Revenue | المبيعات - المرتجعات |
| **COGS** | Cost of Goods Sold | تكلفة المنتجات المباعة |
| **المصاريف** | Expenses | مجموع المصاريف المسجلة |
| **المخزون** | Inventory | الكمية × سعر الشراء |
| **الالتزامات** | Liabilities | ديون الموردين + مستحقات النقل |
| **الربح** | Profit | الإيرادات - COGS - المصاريف |

### ✅ 2. المخزون

**القاعدة:**
- المخزون = الكمية الحالية × سعر الشراء
- المخزون يُعتبر أصل (Asset) ولا يدخل ضمن رأس المال
- عند البيع: يُخفض المخزون بقيمة COGS
- عند الإرجاع: يُعاد COGS للمخزون (يتم تحديث quantity في Product)

**التنفيذ:**
```python
inventory_value = calculate_inventory_value()
# = sum(Product.quantity * Product.buy_price)
```

### ✅ 3. المبيعات

**القاعدة:**
- المبيعات تُسجل كإيراد (Revenue)
- الإيرادات = المبيعات - المرتجعات
- لا يتم خصم COGS أو المصاريف هنا (تُحسب في الربح)

**التنفيذ:**
```python
revenue = calculate_total_revenue()
# = total_sales - total_returns
```

### ✅ 4. المرتجعات

**القاعدة:**
- المرتجعات تُخصم من الإيرادات
- عند الإرجاع: تُعاد تكلفة المنتج (COGS) للمخزون
- COGS للمرتجعات تُخصم من COGS الكلية عند حساب الربح

**التنفيذ:**
```python
returns = calculate_total_returns()
returns_cogs = calculate_returns_cogs()
# عند حساب الربح: net_cogs = cogs_sold - cogs_returned
```

### ✅ 5. المصاريف

**القاعدة:**
- المصاريف تُسجل في حساب مستقل
- لا تؤثر على المخزون
- لا تؤثر على رأس المال مباشرة
- تُطرح من الربح عند حساب صافي الربح

**التنفيذ:**
```python
expenses = calculate_total_expenses()
# = sum(Expense.amount)
```

### ✅ 6. الربح

**القاعدة المحاسبية الصحيحة:**
```
صافي الربح = (الإيرادات - المرتجعات) - (COGS المبيعات - COGS المرتجعات) - المصاريف
```

**أو بشكل مبسط:**
```
صافي الربح = الإيرادات - COGS الصافي - المصاريف
```

**ملاحظة مهمة:**
- الربح لا يُضاف لرأس المال مباشرة
- يُضاف لرأس المال فقط في نهاية الفترة المالية (باستخدام `close_profit_to_capital`)

**التنفيذ:**
```python
net_profit = calculate_net_profit()
# = revenue - net_cogs - expenses
```

### ✅ 7. الالتزامات

**القاعدة:**
- ديون الموردين ومستحقات النقل تُعتبر التزامات (Liabilities)
- لا تؤثر على الربح إلا عند الدفع (تُسجل كمصروف)
- لا تدخل في حساب رأس المال

**التنفيذ:**
```python
supplier_debts = calculate_supplier_debts()
shipping_due = calculate_shipping_due()
```

## ⚠️ قواعد مهمة

### ✅ صحيح:

1. **الإيرادات = المبيعات - المرتجعات** ✅
2. **COGS الصافي = COGS المبيعات - COGS المرتجعات** ✅
3. **الربح = الإيرادات - COGS الصافي - المصاريف** ✅
4. **المخزون يُعتبر أصل ولا يدخل في رأس المال** ✅
5. **الالتزامات لا تؤثر على الربح إلا عند الدفع** ✅

### ❌ خطأ (تم تصحيحه):

1. ❌ حساب الربح بدون خصم المرتجعات
   - **الصحيح:** الربح = (المبيعات - المرتجعات) - COGS - المصاريف

2. ❌ عدم فصل الإيرادات عن المبيعات
   - **الصحيح:** الإيرادات = المبيعات - المرتجعات

3. ❌ عدم خصم COGS المرتجعات
   - **الصحيح:** COGS الصافي = COGS المبيعات - COGS المرتجعات
